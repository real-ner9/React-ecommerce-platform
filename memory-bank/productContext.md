# Product Context

## Purpose
E-commerce platform enabling users to browse, filter, and purchase products online with full order management and admin capabilities.

## Problems Solved
- Product discovery through multi-criteria filtering
- Seamless shopping experience with cart and checkout
- User account management with order history
- Admin product and order management
- Secure authentication and authorization

## User Experience Goals

### Customer Flow
1. Browse products on main page or catalog
2. Filter by category, brand, color, price range, amount
3. View product details
4. Add to cart or favorites
5. Checkout with address input
6. Track orders in profile

### Admin Flow
1. Login with admin privileges
2. Manage products (CRUD operations)
3. Manage categories, brands, colors, amounts
4. Manage homepage slider
5. View and process orders

## Key Features
- Responsive design (mobile/desktop layouts)
- Real-time search functionality
- Pagination for product listings
- Image upload for products
- Email confirmation for registration
- YooKassa payment integration
