# Project Brief

## Project Name
React Shop Vite

## Overview
Full-stack e-commerce application for an online store with product catalog, user authentication, shopping cart, orders, favorites, and payment processing.

## Core Requirements
- Product catalog with filtering (category, brand, color, price, amount)
- User authentication and authorization (JWT-based)
- Shopping cart management
- Order processing and history
- Favorites/wishlist functionality
- Admin panel for product/order management
- Image slider management
- Email confirmation system
- Payment integration (YooKassa)

## Project Structure
```
react-shop-vite/
├── react-shop-frontendd/  # React + Vite + TypeScript frontend
├── react-shop-backendd/   # NestJS + TypeORM + PostgreSQL backend
└── memory-bank/           # Project documentation
```

## Target Users
- Customers: Browse products, manage cart, place orders
- Administrators: Manage products, categories, brands, colors, orders, slider content
